

# AWS Lambda EC2 Automation

This project automates EC2 instance start and stop operations using AWS Lambda and Boto3.

## Features
- Stops instances tagged with Action = Auto-Stop
- Starts instances tagged with Action = Auto-Start



## Technologies
- AWS Lambda
- EC2
- Boto3
- Python
